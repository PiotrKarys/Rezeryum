# REZERYUM — Mapa Struktury Projektu 3.0

Poniższe drzewo prezentuje zoptymalizowany i oczyszczony układ katalogów po asymilacji świata do standardu 3.0.

```text
Rezeryum/
│
├── README.md                      # [BIBLIA] Wprowadzenie, Estetyka, Filary Świata i Napięcia (Hook)
├── 00_struktura.md                # Mapa projektu (ten plik)
├── 01_geografia.md                # Mapa fizyczna, podział na Bastiony, Przedpola, Dzicz
├── 02_system_mocy.md              # Mechaniki Przebudzenia, Esencja, Sploty i 5 Progów Mocy
├── 03_zagrozenia.md               # Bestiariusz, Pęknięcia, Cienie Zbiorowe i Pustoszyciele
├── 04_frakcje.md                  # Gildie, Latarnicy, Polityka i Kulty 
├── 99_raport_spojnosci_3.0.md     # Audyt weryfikujący luki w świecie
│
└── Rasy/                          # [GATUNKI] Katalog dedykowany wyłącznie duszom i ewolucji
    ├── 00_index.md                # [MASTER-INDEX] Tabele ewolucji, dynamika gett, instrukcje szablonu
    ├── aurynn.md                  # Rasa Niebiańskich
    ├── drask.md                   # Rasa Żywiołowych (Linia Żaru)
    ├── ghorran.md                 # Rasa Bestyjnych (Instynkt)
    ├── myrrhan.md                 # Rasa Głębinowych
    ├── orvhan.md                  # Rasa Głębinowych (Pustoż)
    ├── prosterzy.md               # Rasa Głębinowych (Wampiryzm / Krwawe Kontrakty)
    ├── raashari.md                # Rasa Cywilizowanych (Arystokracja Bastionów)
    ├── shenn.md                   # Rasa Żywiołowych (Nekrobotanika)
    ├── solhari.md                 # Rasa Niebiańskich 
    ├── tharumi.md                 # Rasa Cywilizowanych (Mechanika / Runiczni)
    ├── thrakkor.md                # Rasa Głębinowych (Naznaczeni Cieniem)
    ├── velmari.md                 # Rasa Głębinowych (Rój / Kasta Chitynowa i Zarodnikowa)
    ├── voryth.md                  # Rasa Głębinowych (Anomalia Czasu)
    └── vrakai.md                  # Rasa Żywiołowych
```

Ten układ zapewnia techniczny i jednorodny podział. Aspekty absolutne i socjologiczne samego świata znajdują się w rdzeniu katalogu `Rezeryum`, podczas gdy cała biologia i mechanizm przeżyć ewolucyjnych przypisany jest rygorystycznie do katalogu `/Rasy/`.
